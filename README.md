# NFT Indexer

## Description

NFT Indexer is an indexing service that utilize Bull manager to queue transfer events for ERC721 and ERC1155 and store those transfer events and the related ownership records.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
